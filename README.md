**Aurum & Time** is a fully functional e-commerce website for a direct-to-consumer luxury watch and demi-fine jewelry brand. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

> 🌐 **Live Demo:** [aurumandtime.netlify.app](https://aurumandtime.netlify.app)

---

## ✨ Features

### 🛍️ E-Commerce
- **Product Catalog** — Dynamic product loading from a centralized JavaScript database
- **Category Filtering** — Filter jewelry by type (necklaces, bracelets, rings, earrings)
- **Product Detail Pages** — Individual pages with image gallery, features list, and quantity selector
- **Shopping Cart** — Full cart functionality with `localStorage` persistence
- **Checkout Ready** — Prepared for Snipcart, Stripe, or Shopify Buy Button integration
- **Promo Code System** — Built-in discount code engine

### 📊 Inventory Management
- **Stock Tracking** — Real-time inventory quantities with low-stock alerts
- **Admin Dashboard** — Password-protected inventory management panel
- **Visual Stock Status** — Color-coded badges (In Stock / Low Stock / Out of Stock)
- **Quick Restock/Sell** — One-click inventory adjustments
- **Console Commands** — Developer tools for rapid inventory updates

### 🔒 Security
- **Admin Authentication** — Session-based password protection for the dashboard
- **Security Headers** — X-Frame-Options, XSS Protection, Content-Type sniffing prevention
- **Input Validation** — All user inputs sanitized
- **No Sensitive Data** — No API keys or secrets in client-side code

### 🎨 Design
- **Responsive Layout** — Mobile-first design works on all screen sizes
- **Modern Aesthetic** — Minimalist, luxury-focused UI
- **Smooth Animations** — Cart notifications, hover effects, transitions
- **Accessibility** — Semantic HTML, proper contrast ratios, keyboard navigation

### 🚀 Performance
- **Zero Dependencies** — No npm packages, no build step required
- **Static Site** — Deploy anywhere (Netlify, GitHub Pages, Vercel, any server)
- **Fast Loading** — Minimal CSS/JS, optimized images via Unsplash CDN
- **SEO Ready** — Semantic markup, meta tags, sitemap-ready structure
